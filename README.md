# STM32F401 Embedded Drivers

## 📌 Overview
This repository contains embedded driver implementations for STM32F401 microcontroller using register-level programming.

## 🎯 Objective
To understand low-level hardware interaction without using HAL libraries.

## 🧠 Concepts Covered
- GPIO configuration
- UART communication
- Interrupt handling
- Register-level programming

## 🛠️ Tech Stack
- Embedded C
- STM32F401 (ARM Cortex-M4)
- Keil / STM32CubeIDE

## 📂 Project Structure
/src        → Source files  
/include    → Header files  
/docs       → Documentation  

## 🚀 How to Run
1. Open project in STM32CubeIDE / Keil  
2. Flash code to STM32F401 board  
3. Observe output via serial monitor
